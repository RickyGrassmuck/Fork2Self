Welcome

Fork2Self adds a one-click "fork" action to GitHub, GitLab, Gitea/Forgejo, and Bitbucket pages. Forks land in a self-hosted Git instance you control.

This wizard will configure your destination so you can fork right away. You can revisit any of these settings later from the extension's options page.
